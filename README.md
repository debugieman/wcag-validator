# WCAG Analyzer

A web application for validating website accessibility against WCAG (Web Content Accessibility Guidelines) standards.

## Tech Stack

- **Backend**: ASP.NET Core Web API (.NET 10)
- **Frontend**: Angular(21.1.0), Typescipt(5.9.2)
- **Database**: SQLite with Entity Framework Core
- **Architecture**: Clean Architecture (Domain, Application, Infrastructure, Api)
- **Testing**  xUnit, FluentAssertion, Moq
- **BackGround processing** BackgroundService + Channel<Guid>
